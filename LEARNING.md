# DM Learning System

## Overview

This system tracks how you play, what you love, what frustrates you, and how the DM can improve. Every session makes the next one better.

## Files

- `learning-system.json` — Core data structure
- `feedback.md` — Session-by-session notes
- `preferences.md` — Evolving player profile

## How It Works

### After Every Session

The DM asks 3-5 questions:
1. What was your favorite moment?
2. What dragged or bored you?
3. What would you want more/less of?
4. Any rules feel off?
5. How was the pacing?

### What Gets Tracked

**Play Style Detection:**
- Combat vs roleplay ratio you actually enjoy
- How much you engage with NPCs
- Risk tolerance (cautious planner vs reckless hero)
- Puzzle engagement (love/hate/tolerate)
- Leadership style (party face, tactician, lurker)

**Preference Learning:**
- Scene length (quick cuts vs lingering atmosphere)
- Description detail (impressionistic vs granular)
- NPC depth (every guard has a name vs only named characters matter)
- Combat complexity (theater of mind vs tactical grid)
- Story structure (epic narrative vs emergent sandbox)

**Pattern Recognition:**
- Which encounter types energize you
- What causes decision paralysis
- When you check out vs lean in
- Your "tells" (how you signal engagement/boredom)

### DM Adaptations

Based on data, the DM automatically adjusts:

**Narration:**
- More/less sensory detail
- Faster/slower scene transitions
- More/fewer NPC voices
- Combat description density

**Pacing:**
- Shorter/longer exploration segments
- More/fewer random encounters
- Combat round speed
- Time between revelations

**Content:**
- Add more of what you loved
- Cut what bored you
- Adjust difficulty based on risk tolerance
- Introduce NPCs matching your interaction style

**Rules:**
- Strict RAW vs rulings over rules
- Lethality level
- Resource tracking granularity
- XP vs milestone progression

## Feedback Loop

```
Session → Feedback → Analysis → Adjustment → Next Session
```

Each loop makes the game more "yours."

## Learning Triggers

The DM notices and logs:

**High Engagement:**
- "Tell me more about..."
- Asking NPCs personal questions
- Taking notes
- Proactive planning
- Emotional reactions (laughing, cursing, cheering)

**Low Engagement:**
- "Can we just..."
- Checking phone/side conversations
- Rushing through scenes
- Passive following
- "Whatever, I guess we..."

**Decision Patterns:**
- Always/never scout ahead
- Kill vs capture vs negotiate
- Loot everything vs speed run
- Rest constantly vs push forward

## Improvement Log

Major adaptations get logged:

```
Session 3: Player asked for more NPC backstory
→ Added detailed NPC generation system
→ Result: +engagement with tavern scenes

Session 5: Combat felt too slow
→ Switched to theater-of-mind for mooks
→ Kept grid for boss fights only
→ Result: Better pacing, maintained tactical depth

Session 8: Player frustrated by invisible traps
→ Added more environmental clues
→ "Fair" traps with detectable warnings
→ Result: Same danger, more agency
```

## The Goal

Not a perfect DM. YOUR DM. The one who knows that you love political intrigue but hate inventory management. Who gives you exactly enough description to see the scene without drowning you in adjectives. Who challenges you just enough to feel earned victory.

Every session, we get closer.
