# NEW CAMPAIGN SETUP CHECKLIST
## Use This When Starting a New Campaign

---

## BEFORE YOUR FIRST SESSION

### 1. Create Campaign Folder Structure
```bash
/campaigns/[Campaign Name]/
├── chronicle.md              ← Create this FIRST (start with Prologue)
├── sessions/
│   └── session_001.md        (create template)
├── characters_met.md
├── campaign.json
├── maps/
├── images/
├── npcs/
├── treasure/
└── handouts/
```

### 2. Write the Prologue
Before Session 1, write a **2-4 page prologue** that:
- Sets the tone and world
- Introduces the central mystery/conflict
- Establishes the voice (Martin/Abercrombie)
- Makes the reader want to know what happens next
- Contains NO session-specific events yet

**Example opening lines:**
- "The letter arrived on a grey morning, sealed with wax the color of old blood."
- "Trust is what you trade when you have nothing else."
- "The prophecy had been clear. That was the problem."
- "Some debts are written in blood. Others are written in silence."

### 3. Prepare Session Files
In `/campaigns/[Campaign Name]/sessions/`, create:
```markdown
# Session 001

**Date:** [Date]
**Starting Location:** [Location]
**Party Level:** [Level]

## Session Goals
- [ ] Goal 1
- [ ] Goal 2
- [ ] Goal 3

## Events

### Opening Scene
[What starts this session?]

### Key Moments
[Turning points, discoveries, emotional beats]

### Combat Encounters
[Fights, resistances, tricks]

### NPC Interactions
[Who did they meet? What did they learn?]

### Discoveries
[What did they find/learn?]

## Session End
**Ending Location:** [Where did it end?]
**XP Earned:** [Total]
**Treasure Found:** [List]
**Consequences:** [What matters next session?]

[Anything the GM needs to remember?]

---
*Session 001 Log*
```

### 4. Reference the Guides
Keep these open while writing:
- `CHRONICLE_SYSTEM.md` — The full system
- `CHRONICLE_OPERATIONAL_GUIDE.md` — Your voice and style
- `CHRONICLE_WRITERS_CHECKLIST.md` — The workflow

## AFTER EACH SESSION

### Complete This Workflow (Order Matters)

**Step 1: Log Raw Notes** (5 min)
Update `sessions/session_[X].md` with:
- Opening scene
- Key moments (emotional beats)
- Combat (who fought, outcome, cost)
- NPCs (dialogue fragments, reactions)
- Discoveries (loot, secrets, revelations)
- Ending scene
- Consequences for next session

**Step 2: Extract Dialogue** (5 min)
From dialogue system + memory, gather:
- NPC dialogue spoken
- Tone and emotional state
- Relationship shifts
- Any dialogue that surprised you

**Step 3: Write Chronicle Chapter** (60-90 min)
Use `CHRONICLE_WRITERS_CHECKLIST.md`:
- Opening hook
- Setting details
- POV character grounded
- Session events woven naturally
- Internal conflict
- Closing beat
- Target: 2,500-4,000 words

**Step 4: Append to chronicle.md** (5 min)
- Add chapter to correct Book
- Update Table of Contents
- Note session number and date

**Step 5: Reread Entire Chronicle** (20 min)
- Consistency check
- Relationship tracking
- NPC voice consistency
- Arc integrity

## MANAGING MULTIPLE CAMPAIGNS

If you have 2-3 concurrent campaigns:

**Tamoachan Expedition:**
- `/campaigns/Tamoachan Expedition/chronicle.md`
- `/campaigns/Tamoachan Expedition/sessions/`

**Curse of Strahd:**
- `/campaigns/Curse of Strahd/chronicle.md`
- `/campaigns/Curse of Strahd/sessions/`

**New Campaign:**
- `/campaigns/[New Name]/chronicle.md`
- `/campaigns/[New Name]/sessions/`

Each has its own chronicle. Each grows independently. Each tells its own story.

## LONG-TERM ARCHIVING

**When a campaign ends:**
1. Write a final Epilogue (2-3 pages of aftermath)
2. Reread entire chronicle start to finish
3. Count total words (you'll have 50,000-150,000 words)
4. Note it as complete
5. Archive for posterity

**You now have:** A complete fantasy novel of your adventure.

## QUICK REFERENCE: Your Standing Orders

- **After every session:** Chronicle chapter (same day if possible)
- **Before writing:** Review the checklist
- **Always:** Martin's introspection + Abercrombie's cynicism
- **Never:** Generic description, exposition, boring dialogue
- **Always:** Consequences, specific detail, dark humor, character conflict
